Guía de Astro

¿Qué es Astro?

Astro es un framework web moderno para crear sitios web rápidos y optimizados. Se caracteriza por su enfoque en el "Zero JavaScript by default" y su capacidad para trabajar con múltiples frameworks.

Características Principales

  • Componentes Astro (.astro)
  • Islas (Islands Architecture)
  • Zero JavaScript por defecto
  • Integración con múltiples frameworks
  • Optimización automática de assets

Estructura de Archivos

  • src/pages/ - Rutas y páginas
  • src/components/ - Componentes reutilizables
  • src/layouts/ - Plantillas de página
  • public/ - Archivos estáticos
  • src/assets/ - Recursos optimizados
