Gestión de Assets

Optimización de Imágenes

Formatos Recomendados

  • WebP: Mejor compresión, soporte moderno
  • AVIF: Última generación, excelente compresión
  • JPEG: Para fotografías
  • PNG: Para gráficos con transparencia
  • SVG: Para iconos y gráficos vectoriales

Tamaños Recomendados

  • Imágenes de héroe: 1920x1080px
  • Miniaturas: 400x400px
  • Iconos: 24x24px o 32x32px
  • Favicon: 32x32px

Herramientas de Optimización

Squoosh - Optimizador de imágenes de Google TinyPNG - Compresor de imágenes SVGOMG - Optimizador de SVGs Vectorizer - Conversor de imágenes a vector

Implementación en Astro

					
---
import { Image } from 'astro:assets';
import myImage from '../assets/my-image.jpg';
---

<Image
  src={myImage}
  alt="Descripción de la imagen"
  width={800}
  height={600}
  format="webp"
  quality={80}
/>

Estructura de Directorios

					
src/
├── assets/
│   ├── images/
│   │   ├── hero/
│   │   ├── icons/
│   │   └── backgrounds/
│   ├── fonts/
│   └── videos/
└── public/
    └── favicon.ico

Buenas Prácticas

  • Usa nombres descriptivos para los archivos
  • Implementa lazy loading para imágenes
  • Proporciona siempre texto alternativo
  • Utiliza srcset para imágenes responsivas
  • Mantén una estructura de directorios organizada
  • Optimiza los assets antes de subirlos
